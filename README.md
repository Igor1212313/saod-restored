
# SAOD Tests Restored Version NGTU

## 📌 Описание проекта

Данный проект представляет собой восстановленную версию раздела с тестами по дисциплине САОД, использовавшегося на сайте НГТУ.

Оригинальный ресурс был реализован с использованием технологии Adobe Flash и в текущих условиях перестал работать в современных браузерах. В результате раздел тестирования стал недоступен.

В рамках проекта была выполнена реконструкция тестовой части сайта с сохранением структуры, логики и контента.

Проект является альтернативным техническим решением, направленным на восстановление работоспособности образовательного ресурса.

## 🎯 Цель работы

восстановить неработающий раздел тестирования  
обеспечить запуск Flash-тестов в современных браузерах  
сохранить оригинальную структуру сайта  
подготовить решение для интеграции на сервер  

## 🔧 Что было сделано

## 🧠 Как работает система

Каждый тест представляет собой отдельный SWF файл с одним вопросом и логикой проверки.

В оригинальной версии переход выполнялся через JavaScript вызовы из Flash. В современных браузерах данный механизм работает нестабильно, поэтому реализована внешняя навигация.

После ответа пользователь может переключаться между тестами через интерфейс сайта.
