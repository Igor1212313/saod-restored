Инструкция по тестированию

Тесты предназначены для проверки знаний по отдельным темам  теоретического курса.  Тестирование выполняется в режиме "вопрос - ответ",  либо в режиме заполнения  шаблона ответа. 

 

Тесты реализованы на основе flash - технологии и требуют установки на компьютере Macromedia Flash Player  не ниже версии 8.0.

 

 

При выполнении теста строго следуйте всплывающим подсказкам. При завершении тестирования выход из теста выполняется через кнопку "Назад" в меню браузера:
