Визуализаторы предназначены для пооперационной визуализации алгоритмов для красно- черного дерева и алгоритма построения кратчайших путей во взвешенном графе. Визуализаторы обеспечивают автоматическую или пошаговую демонстрацию последовательных действий алгоритмов с комментированием этих действий. Кроме того визуализаторы реализуют режим тренажера, в котором выбор  действий и принятие решений предоставляется обучаемому  под контролем визуализатора. Визуализаторы снабжены справочной системой со сведениями об изучаемых алгоритмах и правилах работы с  визуализаторами. 

Визуализаторы реализованы на основе java - технологии и требуют установки на компьютере Java Plug-in (Sun)  не ниже версии 1.4.0.

 

 

Рекомендуется перед работой с визуализатором красно - черного дерева тщательно изучить теоретические сведения о красно - черном дереве, об итеративных алгоритмах вставки и удаления из красно - черного дерева.

Порядок работы в среде визуализатора.

1. При необходимости установить флажок использования апплетов в браузере IE: Меню Сервис/Свойства    обозревантеля/Дополнительно/Java(Sun)/Use JRE:

 

 

2. Открыть окно визуализатора щелчком по гиперссылке визуализатора. 

3. Ознакомиться с инструкцией работы с визуализатором (кнопка "О программе").

4. Выполнить первоначальную генерацию структуры (кнопка Генерация").

5. Выполнить видоизменение структуры к желаемому виду (кнопки "Вставка", "Удаление", флажок "Видоизменение").6. 6. Задать режим работы визуализатора (кнопка "Настройка") - демонстрацию или тренажер.

7. Выбрать операцию (для RB - дерева) или запустить выполнение алгоритма (для алгоритма Дейкстры).

8. Наблюдать визуально выполнение операций или алгоритма и комментарии визуализатора (в режиме демонстрации)      или выполнять действия в соответствии с указаниями визуализатора (в режиме тренажера).

9. При необходимости просмотреть сведения об алгоритмах (кнопка "Справка").

10. Завершить работы визуализатора с помощью кнопки браузера "Назад".
